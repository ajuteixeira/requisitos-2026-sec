# Requisitos - Sistema ENADE Comentado (SEC)

Repositório acadêmico com os artefatos de requisitos do **Sistema ENADE Comentado (SEC)**.

## Objetivo do projeto

Aplicar a engenharia de requisitos para especificar um sistema web/mobile de questões comentadas do ENADE, visando auxiliar estudantes no preparo para o exame através de materiais organizados e feedback de desempenho.

## Descrição dos arquivos

- **Requisitos/Glossario.md**: Glossário de termos, siglas e expressões utilizados no domínio do ENADE e nas funcionalidades da plataforma.
- **Requisitos/VisaoDemanda.md**: Documento de visão da demanda contendo o objetivo, proposta de valor, partes interessadas, personas e funcionalidades detalhadas.

## Contexto funcional

O sistema proposto atende a estudantes de graduação, professores e coordenadores de curso, com foco em:

- **Banco de questões**: organização de questões oficiais por ano, curso e área de conhecimento;
- **Feedback pedagógico**: disponibilização de gabaritos com explicações detalhadas e fóruns de discussão colaborativos;
- **Prática realista**: realização de simulados com tempo cronometrado para simular a experiência do exame;
- **Monitoramento**: análise de desempenho por área de conhecimento com visualizações gráficas para alunos e coordenadores.
