## Glossário do Projeto

Glossário de termos, siglas e expressões utilizados no Sistema ENADE Comentado (SEC).

| Termo/Sigla                 | Definição                                                                                                                   | Fonte/Observação        |
| --------------------------- | --------------------------------------------------------------------------------------------------------------------------- | ----------------------- |
| **SEC**                     | Sigla para Sistema ENADE Comentado, a plataforma web/mobile objeto deste projeto de especificação.                          | Proposta do Projeto     |
| **ENADE**                   | Exame Nacional de Desempenho de Estudantes. Componente curricular obrigatório e indicador da qualidade do ensino superior.  | INEP / MEC              |
| **INEP**                    | Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira, órgão provedor das questões e gabaritos oficiais.   | Fornecedor de Dados     |
| **Simulado Cronometrado**   | Funcionalidade que permite ao estudante realizar testes com tempo limitado para simular a experiência real do exame.        | Escopo do Sistema       |
| **Gabarito Comentado**      | Explicações detalhadas das respostas para facilitar a compreensão dos conceitos e a correção de erros.                      | Escopo do Sistema       |
| **Fórum de Discussão**      | Espaço para interação entre usuários em cada questão, promovendo o aprendizado colaborativo.                                | Escopo do Sistema       |
| **Formação Geral**          | Componente do exame que avalia a formação ética, social e de cidadania do estudante.                                        | Edital ENADE            |
| **Componente Específico**   | Questões voltadas especificamente para os conteúdos técnicos e habilidades da área de graduação do aluno.                   | Edital ENADE            |
| **UML**                     | Linguagem de Modelagem Unificada, utilizada para representar a estrutura e o comportamento do sistema através de diagramas. | Metodologia (Modelagem) |
| **Estudante Concluinte**    | Público-alvo principal, composto por alunos de graduação que realizarão o exame.                                            | Público-Alvo            |
| **Dashboard de Desempenho** | Interface de análise estatística com visualizações gráficas sobre a evolução do usuário por área de conhecimento.           | Escopo do Sistema       |
